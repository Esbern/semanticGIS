# Project Experiments

This section publishes selected outputs from experiments developed in the local `projects/` folder.

## Publishing rule

Only files placed under `content/` are treated as website content by Quartz.

- Experiment source code and scratch data: `projects/<project_name>/`
- Published notes: `content/projects/`
- Published assets (images, figures, exports): `content/assets/projects/<project_name>/`
